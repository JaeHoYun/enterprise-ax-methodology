# 채워넣기 워크시트 (진단·발굴·게이트)

> 본문의 방법론을 **현장에서 채워 쓰는 워크시트**입니다. 계산용 스프레드시트가 아니라, 항목을 따라 직접 적어 넣는 마크다운 양식입니다.

진단([문서 02](../docs/02-maturity.md))과 유스케이스 발굴([문서 04](../docs/04-portfolio.md)), 파일럿 게이트([문서 03](../docs/03-alternative.md))를 자사 상황에 맞게 채우는 데 사용합니다.

## 사용법

1. 상황에 맞는 워크시트를 엽니다.
2. 빈칸·체크박스를 자사 환경 값으로 채웁니다. 판단 근거가 필요한 칸은 연계 문서의 해당 절을 함께 확인합니다.
3. 채운 워크시트는 팀 내부에 보관하고, 분기마다 또는 파일럿마다 갱신합니다.

## 중요

- 이 워크시트는 **채워넣기 양식**입니다. 자동 계산은 없으며, 항목별로 직접 값을 적고 판단을 기록하는 용도입니다.
- 모든 항목은 작성 시점 기준이며 환경에 따라 달라집니다. 적용 전 [상위 README](../README.md)와 본문으로 재확인하세요.
